Random Values

Random number generation (RNG) produces a random number (crazy right?) This means that a procedure with RNG can return different values even if the parameters (inputs) do not change

CollegeBoard uses RANDOM(A, B), to return an integer between integers A and B. RANDOM(1, 10) can output 1, 2, 3, 4, 5, 6, 7, 8, 9, or 10

In Python, this would be random.randint(A, B), after importing Python's "random" library (import random)

JavaScript's works a little differently, with Math.random() returning a value between 0 and 1.

To match Python and CollegeBoard, you could make a procedure like this

import random 

def Dice(n):
    sum = 0
    while n !=0:
        sum = sum + random.randint(1,6)
        n = n - 1
    return sum
